## opengl low-level 3d rendering api

### [dependencies]:
* **mesa3d** > 
	* sudo pacman -sy mesa
	* sudo xbps-install -s mesa
* **glad** > https://glad.dav1d.de/
* **glfw** > 
	* sudo pacman -sy glfw-x11
	* sudo xbps-install -s glfw glfw-devel
* ...

#### [developed/tested on arch/void linux (using xorg server)]
